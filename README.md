# oftalmo-marketing-segmentation
Segmentação de contas médicas B2B para indústria oftalmológica utilizando K-Means e Data Wrangling.
Para simular o ecossistema de vendas sem ferir a LGPD, os dados deste pipeline foram gerados sinteticamente com base nas distribuições de mercado e prevalências epidemiológicas reais documentadas pelo Vision and Eye Health Surveillance System (VEHSS) e diretrizes da American Academy of Ophthalmology (AAO).
